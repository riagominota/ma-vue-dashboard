This example illustrates the use of CSS to style a <div> to look like a LED indicator. You can use a point's value to set the background color of the LED indicator or to throb the LED on and off. The CSS classes used to accomplish this are quite simple and are shown below if you wish to modify them (rename the classes and put them in your user stylesheet in UI Settings). 

.ma-led {
    width: 30px;
    height: 30px;
    border: 3px solid lightgrey;
    border-radius: 50%;
    background: radial-gradient(rgba(255,255,255,0.5),rgba(0,0,0,0.5));
}

@keyframes throb-opacity {
  50% { opacity: 0.5 }
  100% { opacity: 1 }
}

.ma-throb-opacity {
    animation: throb-opacity 1s ease-in-out infinite;
}
