The Mango UI module uses AngularJS and AngularJS Material Design to make building dashboards easy using simple HTML-like markup. The aim is that you should not have to write any JavaScript. This page will give you a basic introduction to using AngularJS in your markup. Please see W3 Schools for more basic examples.

Use {{ variable }} double curly braces to surround AngularJS expressions/variables that you want inserted into your page. This is called interpolation.

Use ng-init to initialize your variables to any default value. Use the ng-bind attribute to one-way bind the contents of an element to a variable. Use the ng-model attribute to two-way bind a variable with an input. In this example <pre ng-bind="variableName | json"></pre> is used to output a predefined javascript object as json into the pre-formatted element.

The rest of the example pages will introduce you to the Mango UI Components (also referred to as directives) that are available for you to use in your custom pages.
