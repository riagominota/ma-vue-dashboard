Data point tags allow you to add additional tags to the data point over and above the points device name and name. This gives you more flexibility to create complex data structures. Tags use a {key: value} structure e.g.: Site Name: Building 1. A data point can have many different tags but only one value per tag key. Tags can be used to make dynamic watchlists & pages and provide an intuitive way to query and filter data.
