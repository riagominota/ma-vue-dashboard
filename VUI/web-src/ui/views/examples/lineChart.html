We use AmCharts in the UI module but have provided an easy to use component to include charts on your pages.

With <ma-serial-chart> you can input a point and it's values from <ma-point-values>. You can provide your time range and rollup settings to <ma-point-values> via Date Pickers on the page or the Date Bar, then pass the value data to <ma-serial-chart>.

See API Documentation for <ma-serial-chart> and <ma-point-values>.
