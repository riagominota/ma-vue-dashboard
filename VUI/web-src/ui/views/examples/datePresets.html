Use the <ma-date-range-picker> directive to insert a date range preset picker. This enables you to choose from a list of commonly used date ranges, such as Today so far or Previous week. You can set the preset to choose your default date range value. Set the update-interval attribute to have it update automatically.

Use the <ma-date-picker> directive to display a date/time picker that will open in a popup dialog box. Note that you can also add it as an attribute to an existing <input> tag.

ou can link <ma-date-range-picker> and <ma-date-picker> by outputting from and to values from <ma-date-range-picker>, and inputing the values into ng-model attributes on <ma-date-picker>.

See API Documentation for <ma-date-picker> and <ma-date-range-picker>.
