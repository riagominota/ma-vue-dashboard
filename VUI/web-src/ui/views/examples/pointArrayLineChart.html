This is another more advanced example using the <ma-point-query> directive. The list of points outputted from <ma-point-query> is inputted into <ma-point-values> - points property (note plural points property takes an array of points). The combined values are then passed to <ma-serial-chart>.

See API Documentation for <ma-point-query>, <ma-point-values> and <ma-serial-chart>.

Only the first 5 points matching points are shown.
