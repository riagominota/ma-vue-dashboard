This example shows you how to use <ma-events-table> to display a filterable table of events. A large amount of the markup below is for the dropdown controls for setting the filter options. Alternatively you can use <ma-events-table> and hard code the event filters as shown on the api doc for this component.

Add single-point and point-id attributes to display events for just a single Data Point.

See API Documentation for <ma-events-table>.
