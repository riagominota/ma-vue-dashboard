The <ma-point-statistics> directive provides access to historical stats over a time range on a data point.

The object returned by the statistics attribute contains first, last, minimum, maximum, average, integral, sum, & count properties. Each of these will have a value and a timestamp.

If you are interested only in the change in a value between two times you can add the optional first-last="true" attribute to only return the first and last values as an array, then calculate the difference with {{ statsObj[1].value - statsObj[0].value }}.

See API Documentation for <ma-point-statistics>.

The average for the period is {{ statsObj.average.value }} at {{ statsObj.average.timestamp | maMoment:'format':'lll' }}
The max value for the period is {{ statsObj.maximum.value }} at {{ statsObj.maximum.timestamp | maMoment:'format':'lll' }}
