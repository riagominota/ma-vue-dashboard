The <ma-set-point-value> directive will create a form input element to set the value of a point. It can handle numeric, alphanumeric, binary, and multistate point types and will create an appropriate interface element for each point type.

Alternatively you can set the value of a point by calling the setValue method on a point. This function can be called from within a ng-click expression for example.

See API Documentation for <ma-set-point-value> and setValue() method on the Point object.
