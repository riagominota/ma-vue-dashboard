SVG (Scalable Vector Graphics) is a modern image format built for displaying graphics on the web. Unlike jpg/png raster images, SVG will scale up and down without blurring or losing resolution, due to the fact that it is rendered via mathematical vector based instructions in it's XML syntax.

SVG graphics can be designed in a software such as Adobe Illustrator or Inkscape. Inkscape is free and open-source option for editing vector graphics. With your graphic software or even a text editor you can design your vector images and give specific IDs or classes to elements in your graphic that you want to interact with.

In this example we have an SVG graphic, status-panel.svg that has an #indicator element and a #spinner element. Using the <ma-svg> component you can pass in the path to the image into the ng-include attribute. Then use divs with ma-selector to select an element within the SVG graphic. Any valid CSS selector can be used, in this case we are selecting by ID of an element. Once selected you can apply attributes such as ng-style, ng-class, and ng-click.

Here we have the #indicator element's fill color set by the value of a multi status data point. We also have the #spinner element within the SVG linked to CSS spin classes that can be toggled to reverse spin direction.
