Adaptive Layouts

We have added several example layouts that demonstrate the power of Angular Material's Adaptive Layout API. With no need for complex CSS or media queries, and using just simple HTML attributes, you can create advanced flexbox layouts. There are many features to Angular Material's layout system including flex attributes, page size breakpoints, layout row/column, and layout wrapping. It is worth visiting the official documentation to get the full benefit of having this powerful layout system available in your Mango pages.

You can use these as starting points for a creating an adaptive and responsive layout within your pages. Click New Page From Layout to create a new page using the layout shown. You can then add in Mango UI Components to pull in and display data.
