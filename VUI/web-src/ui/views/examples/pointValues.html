The <ma-point-values> returns an array of historical values on a data point to its values attribute. In this example <ma-point-values> is passed to & from values from the Date Bar. Additionally, rollup & rollup-interval is inputted from the Date Bar.

The resulting array of data is stored in point1Values. To display this data we are using a table and ng-repeat to loop through the results and stamp out a table row for each value in the array. Angular's ng-repeat is a fundamental angular directive used to loop through items in an array and display an HTML template for each item. You will have access to properties of the item if you are looping through an array of objects and can display them within the repeated template.

See API Documentation for <ma-point-values>.
