This example shows how to plot a data point for a whole week, splitting each day into a separate series so you can compare the values for each day.

The trick is to use multiple <ma-point-values> directives combined with the moment filter to get values for each day individually. Each set of daily values is inputted into a series-X-values attribute of of <ma-serial-chart>. Then series-X-title is used to manually set titles for each series and the time-format attribute is used to only show the hourly time on the x-axis.

See API Documentation for <ma-serial-chart>, <ma-point-values>, <ma-now> and momentFilter .
