The <ma-heat-map> component displays a heat map of data point values, useful for visualizing trends in point values over a large period of time. A rollup of 15 minutes, 30 minutes or 1 hour should be used when obtaining point values for use with this component.

Adjust the date bar time period a long period such as 3 months for best results.

See API Documentation for <ma-heat-map> and <ma-point-values>.
