There is a watchdog service which periodically pings an API endpoint to check if the Mango API is up and if the user is logged in.

The mangoWatchdog object is on the root scope, you can check mangoWatchdog.apiUp to see if the API is up and mangoWatchdog.loggedIn to see if the user is logged in. There is also a more detailed status string available at mangoWatchdog.status.

See API Documentation for the mangoWatchdog service.

The watchdog status is .
Failure Connectivity to the Mango API has been lost.
Good Connection to the Mango API is OK.
Good Connection to the Mango API is OK and the user is logged in.
