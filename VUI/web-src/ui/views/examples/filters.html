We can use AngularJS filters to format interpolated values and manipulate values. Filters in the HTML always follow the {{ expression | filter }} syntax. The standard AngularJS filters are available such as number and currency as well as custom filters included with the UI module.

Custom filters include moment and duration for using Moment.js to manipulate dates and durations.

See API Documentation for moment and duration filters.

The raw point value is {{myPoint.value}} at epoch timestamp {{myPoint.time}}.

The formatted point value is {{myPoint.value | number:2}} at time {{myPoint.time | maMoment:'format':'ll LTS'}}.

The price is {{price | currency}}.

The time now as an ISO formatted string is {{'now' | maMoment:'toISOString'}}.

{{timeTaken}}ms is {{timeTaken|maDuration:'humanize'}}

The first day of the month was a {{'now' | maMoment:'startOf':'month' | maMoment:'format':'dddd'}}.

Three days from now is {{'now' | maMoment:'add':3:'days' | maMoment:'format':'LLL'}}.
