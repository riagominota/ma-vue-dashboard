The <ma-data-source-list> directive displays a list of Mango data sources. The selected data source will be output to the variable specified by the ng-model attribute.

The <ma-device-name-list> directive displays a list of Mango device names. The selected device name will be output to the variable specified by the ng-model attribute.

There are also <ma-data-source-query> and <ma-device-name-query> directives which output an array of data source objects and device name strings respectively.

See API Documentation for <ma-data-source-list>, <ma-device-name-list>, <ma-data-source-query> and <ma-device-name-query>.
