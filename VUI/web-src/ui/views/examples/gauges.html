<ma-gauge-chart> will display a gauge/dial chart. Several properties have been exposed via attributes on the component, allowing you to set colors and ranges of multiple bands, and style the arrow.

You must give the gauge a width and height, and you must use <ma-get-point-value> to provide a point value to <ma-gauge-chart>.

See API Documentation for <ma-gauge-chart>.
