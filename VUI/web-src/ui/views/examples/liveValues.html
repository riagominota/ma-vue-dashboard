This example shows the differences between <ma-point-value> and <ma-get-point-value>.

The <ma-point-value> directive will render the live value and other data from the data point. You can optionally supply the display-type attribute to get the unit converted value or data/time of the update.

The <ma-get-point-value> will provide a point variable (named 'myPoint' in this example) that you can use in the scope of your angular app. Live values can be displayed as text within your HTML by using {{myPoint.value}} or {{myPoint.renderedValue}} syntax.

See API Documentation for <ma-point-value> and <ma-get-point-value>.





Using AngularJS interpolation

Note: You need to have a <ma-get-point-value> tag for these to work.
