If you do not want to create pages within this app you can create your own copy of a dashboard. Note: there is much more work involved in creating a custom dashboard and we highly recommend using the built in page system. It is amazing what you will be able to build using just the Mango UI Components, Zero JavaScript, and the built in CMS! Using the built in CMS is the method we will be continuing to support and will evolve over time with even more features.

You will need to work over FTP if Mango is live on a server or in a Code Editor if working on a local machine. The advantage is you can write your own JavaScript controllers and directives and extend our framework with your own AngularJS components. To create your own dashboard, copy one of the templates into a new folder in Mango/overrides/web/modules/mangoUI/web and rename the HTML file to index.html.

There is also a complete Material Design admin template similar to this web app, with a menu which you can customize using JavaScript.

Once you have created a folder in overrides, your dashboard will be accessible at /user-ui/YOUR-FOLDER-NAME.

You will then need to update the loader script at the bottom of the index.html file. Replace require(['mangoUIModule/adminTemplate/app']); with require(['mangoUIModule/YOUR-FOLDER-NAME/app']);.

Once you have a working template you can start playing around with the live examples and copy and paste them into your template HTML file.
