You can use this syntax for <ma-point-value>, <ma-get-point-value> & <ma-point-values>.

The XID of the data point you want to display is manually passed as a string to the point-xid attribute.

Note, this example uses a data point XID from the example virtual data sources, you can import this JSON file or enter your own XIDs.

See API Documentation for <ma-point-value> and <ma-get-point-value>.
